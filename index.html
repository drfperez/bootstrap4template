AUDI CARS: Data and sustainability

This website contains information about AUDI cars, and interactive activities:

  • Three mathematical problems using the MathJax library for formulas.
  • A ten multiple choice questions test about the car data.
  • An image map made with GIMP, that shows the parts of an AUDI car.

Here, as an introduction, I included a H5P presentation about using R language and RStudio libraries to analyse the data of the cars.
